## Usage

### Nix-darwin

1. Enter devshell running `nix develop`
2. Switch to darwin configuration by running `nh darwin switch .`

### Home-manager

1. Enter devshell by running `nix develop`
2. Switch to home-manager configuration by running `nh home switch .`
